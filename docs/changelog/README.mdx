## How Changes Are Tracked

When documentation is updated, our team creates changelog entries that describe:
- What changed
- Why it changed
- Where to find the updated documentation

These entries are then aggregated and published as comprehensive changelogs, making it easy to stay up-to-date with documentation changes.
