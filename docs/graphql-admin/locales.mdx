# GraphQL Admin: Localization overview

## Introduction

The GraphQL Admin API's allows merchants to add and remove languages and manage language settings per each sales channel. 


## Getting started

## Example requests
This section contains examples of GraphQL Admin API local mutations that you can send with the following HTTP configuration.

```http filename="GraphQL Admin API HTTP configuration" showLineNumbers copy
POST https://api.bigcommerce.com/accounts/{{store_hash}}/graphql
X-Auth-Token: {{access_token}}
Accept: application/json
Content-Type: application/json
```
### Query channel languages
The following query returns details about a channel's language. 

<Tabs items={['Request', 'Response']}>
  <Tab>
    ```graphql filename="Example query: Get channel language" showLineNumbers copy
    query getSupportedLocales {
      store {
          channelId
          locale {
            node {
              code
              status
              isDefault
            }
          }
   

### Setting a default language to a channel

The following mutation sets a channel's language. You can specify a language as default and active/inactive.

<Tabs items={['Request', 'Response']}>
  <Tab>
    ```graphql filename="Example query: Set language" showLineNumbers copy
    mutation {
      locale {
         updateLocale(input: {
            channelId: "bc/store/channel/1",
               code: "abc"
                  }) {
                  locale {
                      code
                      status
                        }
                       errors {
                        ... on Error {
                        message
                            }
                        }
                    }
                }
            }

### Manage language settings 

The following query returns details about a channel's language settings. You can specify a language as default and active/inactive.

<Tabs items={['Request', 'Response']}>
  <Tab>
    ```graphql filename="Example query: Get language settings" showLineNumbers copy
    query {
      store {
        locales(input: { channelId: "bc/store/channel/1"}) {
          edges {
            node {
              code
              status
              isDefault
            }
          }
        }
      }
    }
    ```
