
## 6.0 Fulfill order as a merchant

When building a _Buy Online, Pick up in Store_ solution, merchants can fulfill orders for pickup using the existing [Orders V2 API](/docs/rest-management/orders).

To fulfill an order, send a request to the [Create an order](/docs/rest-management/orders#create-an-order) endpoint.
