
## 3.0 Browse and discover as a shopper

When building a _Buy Online, Pick up in Store_ solution, you can create custom experiences for shoppers that support _Buy Online, Pick up in Store_.

There are two ways to access data on the storefront:
- REST Storefront API
- GraphQL Storefront API

Over time, BigCommerce will be investing more in the GraphQL Storefront API, so all future enhancements to support _Buy Online, Pick up in Store_ will be made against the GraphQL Storefront API.

### 3.1 REST Storefront API

#### 3.1.1 Find available pickup options with stock available 

To show pickup options to customers on a storefront page (e.g. the product detail page), send a request to the [Find available pickup options](/docs/rest-storefront/pickup-options#find-available-pickup-options) endpoint.
