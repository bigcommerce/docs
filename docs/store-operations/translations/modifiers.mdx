# Translations for Modifiers (Beta)

<Callout type='info'>
	The Translations Admin GraphQL API is currently available on Catalyst
	storefronts only.
</Callout>

<Callout type='warning'>
	Modifier translations are currently in development. This documentation will be updated when the feature becomes available.
</Callout>

Modifier translations will allow you to translate product modifier content displayed to customers, including:

- Modifier names
- Modifier option values
- Modifier descriptions

## Coming Soon

This feature is currently under development. Check back soon for updates on modifier translation capabilities.
