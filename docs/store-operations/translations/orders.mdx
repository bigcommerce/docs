# Translations for Orders (Beta)

<Callout type='info'>
	The Translations Admin GraphQL API is currently available on Catalyst
	storefronts only.
</Callout>

<Callout type='warning'>
	Order translations are currently in development. This documentation will be updated when the feature becomes available.
</Callout>

Order translations will allow you to translate order-related content displayed to customers, including:

- Order status messages
- Order confirmation content
- Order email templates
- Order-related notifications

## Coming Soon

This feature is currently under development. Check back soon for updates on order translation capabilities.
