# Translations for Promotions (Beta)

<Callout type='info'>
	The Translations Admin GraphQL API is currently available on Catalyst
	storefronts only.
</Callout>

<Callout type='warning'>
	Promotion translations are currently in development. This documentation will be updated when the feature becomes available.
</Callout>

Promotion translations will allow you to translate promotion-related content displayed to customers, including:

- Promotion names
- Promotion descriptions
- Promotion banners
- Promotion messages

## Coming Soon

This feature is currently under development. Check back soon for updates on promotion translation capabilities.
