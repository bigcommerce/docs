# Catalyst version 1.3.7 Release Notes

Catalyst v1.3.7 release addresses a **security vulnerability ([CVE-2025-55184, CVE-2025-55183](https://react.dev/blog/2025/12/11/denial-of-service-and-source-code-exposure-in-react-server-components))** that affects React Server Components.

<Callout type='warning'>
	**Important**: This release provides an additional security patch for the same
	CVEs addressed in [1.3.6](/docs/storefront/catalyst/release-notes/1-3-6). If
	you upgraded to `1.3.6`, you should upgrade to `1.3.7` to receive the latest
	security fixes.
</Callout>

## Key Changes

- **Next.js 15.5.9**: Upgraded from Next.js 15.5.8 to 15.5.9
- **React 19**: Upgraded to React 19.1.4 and React DOM 19.1.4

## Migration Guide

<Steps>
### Update Dependencies

Update dependencies in your `package.json`:

```json
{
	"dependencies": {
		"next": "15.5.9",
		"react": "19.1.4",
		"react-dom": "19.1.4"
	},
	"devDependencies": {
		"@next/bundle-analyzer": "15.5.9",
		"eslint-config-next": "15.5.9"
	}
}
```

### Run `pnpm install`

```bash
pnpm install
```

<Callout type='info'>
	**Note**: `next` will automatically update your `tsconfig.json` file.
</Callout>

</Steps>
