# checkbox

Used to toggle on theme settings (boolean value).

```json filename="checkbox: theme settings" showLineNumbers copy
{
  "type": "checkbox",
  "label": "i18n.HideBreadcrumbs",
  "force_reload": true,
  "id": "hide_breadcrumbs"
},
```

Required fields: `type`, `label`, and `id`.

Optional field: `force_reload`.
