# color

Used to set color value on theme setting (hex value).

```json filename="color: theme settings" showLineNumbers copy
{
  "type": "color",
  "label": "i18n.BannerBackground",
  "id": "color-primary"
},
```

Required fields: `type`, `label`, and `id`.
